# 🎧 AuraTune

### Feel Every Beat

AuraTune is a modern music website built using HTML, CSS, and JavaScript. It features a visually appealing user interface and an interactive music player that allows users to play and pause audio seamlessly.

## ✨ Features

* Interactive Play/Pause Music Control
* Clean and Modern User Interface
* Responsive Layout
* Smooth User Experience
* Lightweight and Easy to Use

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript

## 🚀 Getting Started

1. Clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.
4. Click the play button to enjoy the music.

## 🎯 Purpose

This project was developed to demonstrate front-end web development skills using HTML, CSS, and JavaScript while creating an engaging and aesthetic music website experience.
